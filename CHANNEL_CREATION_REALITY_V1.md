# Telegram Channel Factory — Channel Creation Reality v1

## 1. Purpose

Этот документ фиксирует реалистичный сценарий создания Telegram-канала в рамках продукта.

## 2. Core product truth

В MVP нельзя строить продукт на предположении, что бот полностью магически создаёт и настраивает канал без участия пользователя.

## 3. MVP approach

Реалистичный подход v1:
- пользователь при необходимости создаёт канал вручную
- бот даёт guided instructions
- бот помогает подключить канал к продукту
- после подключения продукт берёт на себя дальнейшее управление контентным пайплайном

## 4. Product rule

Создание канала и подключение канала — это разные сценарии и их нельзя смешивать в одну неясную магию.

## 5. Decisions fixed by this doc

Этим документом фиксируется:
- MVP опирается на guided connection flow
- полностью автономное создание канала не является обязательным условием v1
- основной акцент продукта — управление каналом после подключения
