# Telegram Channel Factory — Release Docs v1

## 1. Purpose

Этот документ фиксирует требования к release-ready документации.

## 2. Goal

Должна существовать документация, по которой можно понять, как установить, настроить, запустить и использовать продукт.

## 3. Must-have docs

- README
- env setup guide
- local run guide
- bot setup guide
- channel connection guide
- test run guide
- deploy guide

## 4. MVP rule

Документация должна быть практичной и ориентированной на запуск, а не формальной.

## 5. Decisions fixed by this doc

Этим документом фиксируется:
- release docs обязательны перед выпуском
- документация должна закрывать path от установки до первого рабочего сценария
- bot/product usage docs так же важны, как backend docs
