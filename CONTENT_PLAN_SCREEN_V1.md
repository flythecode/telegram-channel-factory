# Telegram Channel Factory — Content Plan Screen v1

## 1. Purpose

Этот документ фиксирует экран `Контент-план` для MVP v1.

## 2. Goal

Экран должен позволять пользователю создавать, просматривать и обновлять контент-план канала.

## 3. Core actions

Пользователь должен уметь:
- сгенерировать контент-план
- посмотреть список задач / идей
- перегенерировать план
- добавить идею вручную
- перейти к созданию черновиков

## 4. UX rule

План должен восприниматься не как абстрактный JSON, а как рабочий список тем и задач для канала.

## 5. Decisions fixed by this doc

Этим документом фиксируется:
- экран `Контент-план` входит в MVP
- контент-план является отдельным управляемым объектом продукта
- пользователь должен иметь возможность обновлять план после запуска канала
