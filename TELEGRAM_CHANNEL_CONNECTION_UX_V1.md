# Telegram Channel Factory — Telegram Channel Connection UX v1

## 1. Purpose

Этот документ фиксирует UX подключения реального Telegram-канала к проекту в MVP v1.

## 2. Goal

Подключение канала должно быть понятным, пошаговым и не требовать от пользователя догадок о правах и действиях.

## 3. MVP flow

1. пользователь указывает канал
2. бот объясняет, как добавить его в админы
3. пользователь подтверждает шаг
4. система проверяет возможность публикации / доступ
5. бот показывает результат проверки

## 4. UX rule

Если подключение не удалось, пользователь должен получить не абстрактную ошибку, а понятное объяснение, что исправить.

## 5. Decisions fixed by this doc

Этим документом фиксируется:
- UX подключения канала является отдельным критическим flow
- подключение должно быть guided, а не скрытым
- ошибки подключения должны быть понятными и исправимыми
