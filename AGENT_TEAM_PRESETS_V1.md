# Telegram Channel Factory — Agent Team Presets v1

## 1. Purpose

Этот документ фиксирует preset-команды ИИ-агентов для MVP v1.

Задача preset'ов:
- дать пользователю быстрый старт без сложной ручной настройки
- стандартизировать типовые команды под разные сценарии
- сохранить возможность дальнейшего редактирования после выбора preset

---

## 2. Product rule

В MVP v1 пользователь должен выбирать не из бесконечной конфигурации, а из простых стартовых шаблонов:
- 3 агента
- 5 агентов
- 7 агентов

После выбора preset пользователь может:
- включать и выключать агентов
- менять роли
- менять prompts
- адаптировать команду под свой канал

---

## 3. Preset list

### Preset A — `starter_3`
**Название:** Быстрый старт
**Для кого:** простой канал, быстрый запуск, минимум сложности
**Количество агентов:** 3

Состав:
1. Strategist
2. Researcher
3. Writer

### Preset B — `balanced_5`
**Название:** Сбалансированная команда
**Для кого:** основной рекомендуемый режим MVP
**Количество агентов:** 5

Состав:
1. Strategist
2. Researcher
3. Writer
4. Editor
5. Publisher

### Preset C — `editorial_7`
**Название:** Полная редакция
**Для кого:** более сложный контент, качественный контроль, медиа-подход
**Количество агентов:** 7

Состав:
1. Strategist
2. Researcher
3. Writer
4. Editor
5. Fact-checker
6. Reach Optimizer
7. Publisher

---

## 4. Preset behavior rules

1. Preset создаёт стартовую структуру команды.
2. Preset не является жёсткой схемой навсегда.
3. Пользователь может менять состав после выбора preset.
4. В UI preset должен объясняться простыми словами, а не техническими терминами.
5. `balanced_5` должен быть рекомендован как default choice в MVP.

---

## 5. UX presentation

В Telegram UI preset'ы показываются так:
- `3 агента — Быстрый старт`
- `5 агентов — Сбалансировано`
- `7 агентов — Полная редакция`

Под каждым вариантом должен быть короткий смысл:
- быстро
- сбалансированно
- максимально полно

---

## 6. Data model guidance

Каждый preset должен иметь:
- `code`
- `title`
- `agent_count`
- `description`
- `roles_json`
- `is_recommended`
- `is_active`

---

## 7. Decisions fixed by this doc

Этим документом фиксируется:
- в MVP есть 3 preset-команды
- базовый рекомендуемый preset — `balanced_5`
- preset'ы ускоряют setup, но не ограничивают последующее редактирование
- 3–7 агентов остаются продуктовой нормой v1
