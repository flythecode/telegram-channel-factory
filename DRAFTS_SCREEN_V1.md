# Telegram Channel Factory — Drafts Screen v1

## 1. Purpose

Этот документ фиксирует экран `Черновики` для MVP v1.

## 2. Goal

Экран должен дать пользователю удобное место для ревью и управления draft'ами перед публикацией.

## 3. Core actions

Пользователь должен уметь:
- смотреть список черновиков
- открывать конкретный draft
- approve draft
- reject draft
- редактировать текст
- перегенерировать draft
- отправлять draft в очередь публикации

## 4. Product rule

Черновик — центральный объект контроля качества между генерацией и публикацией.

## 5. Decisions fixed by this doc

Этим документом фиксируется:
- экран `Черновики` обязателен для MVP
- approve/reject flow обязателен для MVP
- ручное редактирование draft'а входит в MVP
