# Telegram Channel Factory — Publications Queue Screen v1

## 1. Purpose

Этот документ фиксирует экран `Публикации / Очередь` для MVP v1.

## 2. Goal

Экран должен позволять пользователю видеть статус публикаций и управлять очередью отправки в Telegram-канал.

## 3. Core actions

Пользователь должен уметь:
- видеть queued / sending / sent / failed
- открывать карточку публикации
- опубликовать сейчас
- изменить время публикации
- остановить публикацию
- удалить из очереди

## 4. Product rule

Очередь публикаций должна быть управляемой, а не полностью скрытой автоматикой.

## 5. Decisions fixed by this doc

Этим документом фиксируется:
- экран публикаций обязателен для MVP
- пользователь сохраняет контроль над публикацией даже при автоматизации
- queue management — важная часть Telegram admin panel
