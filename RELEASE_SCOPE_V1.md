# Telegram Channel Factory — Release Scope v1

## 1. Purpose

Этот документ фиксирует релизный скоуп MVP v1.

## 2. Goal

Нужно жёстко отделить то, что входит в первый выпуск, от того, что уходит в post-MVP, чтобы продукт не расползался.

## 3. In scope

- Telegram-first bot UI
- setup wizard
- channel connection flow
- presets 3/5/7
- editable agent team
- content plan / tasks / drafts / publications
- manual / semi-auto / auto modes
- post-setup management

## 4. Out of scope

- enterprise collaboration
- billing
- deep analytics
- omnichannel
- advanced graph orchestration

## 5. Decisions fixed by this doc

Этим документом фиксируется:
- MVP должен быть ограничен и собран в чёткий продуктовый контур
- всё второстепенное переносится за пределы релиза
- release scope управляет приоритетами разработки
