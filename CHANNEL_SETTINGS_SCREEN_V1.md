# Telegram Channel Factory — Channel Settings Screen v1

## 1. Purpose

Этот документ фиксирует экран `Настройки канала` для MVP v1.

---

## 2. Goal

Экран должен позволять менять базовые параметры канала после создания без участия разработчика и без редактирования кода.

---

## 3. Editable settings

В MVP экран должен поддерживать изменение:
- темы / ниши
- языка
- tone of voice
- типа контента
- частоты публикаций
- рубрик
- approval policy
- operation mode

---

## 4. UX rule

Каждый параметр редактируется через короткий отдельный сценарий:
- увидеть текущее значение
- выбрать новое значение
- подтвердить изменение

---

## 5. Expected result

После сохранения настроек новые задачи и новые генерации должны использовать обновлённую конфигурацию.

---

## 6. Decisions fixed by this doc

Этим документом фиксируется:
- post-setup редактирование настроек обязательно входит в MVP
- экран `Настройки канала` является базовым элементом channel dashboard
- пользовательский контроль над стратегией канала — обязательная функция продукта
