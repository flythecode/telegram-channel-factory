# Telegram Channel Factory — Channel Creation Wizard v1

## 1. Purpose

Этот документ фиксирует структуру wizard-сценария создания канала в Telegram-боте для MVP v1.

---

## 2. Goal

Wizard должен позволить пользователю быстро запустить проект канала с минимальным трением и без перегруженной анкеты.

---

## 3. Wizard steps

1. Старт создания проекта
2. Название проекта
3. Тема / ниша
4. Язык
5. Цель канала
6. Тип контента
7. Частота публикаций
8. Summary и подтверждение
9. Выбор команды агентов
10. Подключение Telegram-канала
11. Первый запуск проекта

---

## 4. UX rules

- один шаг = один вопрос
- максимум кнопок, минимум длинного текста
- всегда есть `Назад` и `Отменить`
- после завершения wizard пользователь сразу попадает в рабочий проект

---

## 5. State persistence rule

Если пользователь прерывает wizard, система должна по возможности сохранять промежуточный прогресс и позволять продолжить позже.

---

## 6. Decisions fixed by this doc

Этим документом фиксируется:
- setup делается через пошаговый wizard
- wizard короткий и кнопочный
- wizard заканчивается не тупиком, а переходом в рабочий dashboard канала
