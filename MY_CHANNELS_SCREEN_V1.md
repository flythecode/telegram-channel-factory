# Telegram Channel Factory — My Channels Screen v1

## 1. Purpose

Этот документ фиксирует экран `Мои каналы` для MVP v1.

---

## 2. Goal

Экран должен давать пользователю единый список всех его проектов/каналов и быстрый вход в управление каждым из них.

---

## 3. What the screen shows

Для каждого элемента списка должны отображаться:
- название проекта
- название / идентификатор канала
- текущий режим работы
- краткий статус

---

## 4. Main actions

Пользователь должен иметь возможность:
- открыть карточку канала
- создать новый канал
- вернуться в главное меню

---

## 5. Empty state

Если каналов нет, экран должен показывать:
- короткое объяснение
- кнопку `Создать первый канал`
- кнопку `Как это работает`

---

## 6. Decisions fixed by this doc

Этим документом фиксируется:
- `Мои каналы` — основной вход в повторное использование продукта
- список каналов обязателен для MVP
- через этот экран пользователь возвращается к управлению после setup
