# Telegram Channel Factory — Agents Screen v1

## 1. Purpose

Этот документ фиксирует экран `Агенты` для MVP v1.

## 2. Goal

Экран должен давать пользователю понятное управление командой ИИ-агентов внутри конкретного канала.

## 3. Core actions

Пользователь должен уметь:
- смотреть текущий состав команды
- открывать карточку агента
- менять preset
- включать/выключать агента
- менять роль
- менять базовые настройки агента

## 4. UX rule

Экран должен показывать не технические сущности, а редакционные роли с кратким человеческим объяснением.

## 5. Decisions fixed by this doc

Этим документом фиксируется:
- экран `Агенты` обязателен для MVP
- команда агентов должна быть управляемой после запуска
- настройка агентов — часть повседневного управления каналом
