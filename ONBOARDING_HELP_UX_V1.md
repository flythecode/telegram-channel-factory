# Telegram Channel Factory — Onboarding & Help UX v1

## 1. Purpose

Этот документ фиксирует onboarding/help слой продукта для MVP v1.

## 2. Goal

Пользователь должен понимать, что делает продукт, как пройти setup, как подключить канал и как вернуться к управлению позже.

## 3. What must exist in MVP

- стартовое объяснение на `/start`
- экран `Как это работает`
- экран `Помощь`
- контекстные подсказки на сложных шагах
- понятные тексты при ошибках подключения и публикации

## 4. UX rule

Подсказки должны быть короткими, конкретными и написанными человеческим языком.

## 5. Decisions fixed by this doc

Этим документом фиксируется:
- onboarding/help UX входит в MVP
- продукт не должен требовать внешней инструкции для базового использования
- help-сценарии особенно важны на шагах подключения канала и смены режимов
